# tastManager-springboot-backEnd

Manage Task, Employee and Chefs:
 - create task as chef
 - assign task as chef to employee
 - assign task as chef to chef
 - update tast as chef
 - delete task as chef

Framework:
 - springboot

Tests:
- integration tests with JUnit5

Builded with:
 - Maven
CI:
 - Github CI

Deployement:
 - Heroku
